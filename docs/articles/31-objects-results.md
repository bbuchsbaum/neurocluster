# Objects and results

## cluster4d_result

Fields: `clusvol`, `cluster`, `centers`, `coord_centers`, `n_clusters`,
`method`, `parameters`, `metadata`.

## Print/plot/summary

- `print(x)`: method, counts, cluster size stats
- `summary(x)`: parameters, cluster size distribution, ranges
- `plot(x)`: axial/sagittal/coronal slices
