# Meta Clustering for Cluster Results (S4 Generic)

The meta_clust generic function is used to define methods for different
classes of input objects.

## Usage

``` r
meta_clust(x, cuts, ...)
```

## Arguments

- x:

  A clustering result, typically an object of class `"cluster_result"`.

- cuts:

  The number of cluster cuts to consider.

- ...:

  Additional arguments passed to methods.

## Value

Depends on the method called.
